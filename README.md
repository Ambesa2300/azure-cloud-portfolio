### Project 2 — Secure Storage & Cost Optimization
**Focus:** Secure and cost-efficient data management

- Static website hosting (Blob Storage)
- Private containers secured with SAS tokens
- Lifecycle management rules
- Cost-aware architecture

➡️ See: `project-02-storage.pdf`

### Project 3 — Secure Network Architecture
**Focus:** Least-privilege networking design

- Virtual Machine deployed without public IP
- Secure access via Azure Bastion
- Network Security Groups (NSG) with restricted inbound rules
- Controlled outbound connectivity (HTTP/HTTPS only)

➡️ See: `project-03-secure-network.pdf`
